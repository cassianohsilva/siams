Sobre o SIAMS

O SIAMS é um programa fácil de usar feito com a finalidade de ajudar principalmente a comunidade acadêmica a aprender a lingaguem de programação MIPS. Ele possui interface amigável e fácil de usar.
SIAMS é composto por um editor onde deve-se inserir o código em MIPS, uma tabela que mostra o valor dos registradores e também uma aba onde é possível ver os endereços da memória onde estão sendo utilizados pelo código do programa

Sobre o SIAMS

O nome SIAMS é um acrônimo e significa Siams Is A Mips Simulator

Sobre o SIAMS

O usuáro pode abrir um arquivo do disco rígido clicando em Abrir.

Ao inserir o código e clicar em Montar, é gerado um código binário que é inserido na memória.

Ao clicar em Executar,todo o código é compilado é executado de uma vez.

Se você clicar em Step o código executado instrução por instrução, e é possivel ver o valor dos registradores sendo alterados na tabela do lado direito.

  • Atalhos

    • Ctrl + Espaço = Abre uma janela de atalhos

    Ctrl + O = Abrir um arquivo

    Ctrl + H = Abre a janela do tutorial

    Ctrl + P = Mostra as instruções existentes no programa

  • Instruções Básicas

    • add = Istrução de soma

    sub = Instrução de subtração

    lw = Dados da memória para o registrador

    sw = Dados do registrador para a memória

    and = Instrução lógica AND bit a bit

    or = Instrução lógica OR bit a bit

    nor = Instrução lógica NOR bit a bit

    andi = Instrução lógica AND bit a bit com contante

    ori = Instrução lógica OR bit a bit com constante

    sll = Deslocamento bit a bit à esquerda por constante

    srl = Deslocamento bit a bit à direita por constante

  • Instruções Básicas

    • beq = Teste igualdade e desvia

    bne = Teste desigualdade e desvia

    j = Desvia para o endereço de destino

    jr = Para retorno de procedimento

    jal = Para chamada de procedimento

    Essas são algumas das instruções básicas úteis em MIPS

  • Exemplo 1:

                     Loop: sll $t1,$s3,2  #Reg.temporario $t1= 4*i
                        add $t1,$t1,$s6   # $t1 = endereço de save[i]
                        lw $t0,0($t1)     # Reg. temporário  $t0 = save[i]
                        bne $t0,$s5,Exit  # Vai para Exit se save[i] != k
                        addi $s3,$s3,1    # i = i+1
                        j Loop            # vai para loop
                     Exit:
  • Exemplo 2:

                     fact:
                        addi $sp,$sp,-8   #ajusta pilha para 2 itens
                        sw   $ra,4($sp)   #salva o endereço de retorno
                        sw   $a0,o($sp)   #salva argumento n
                        slti $t0,$a0,1    #teste para n<1
                        beq  $t0,$zero,L1 #se n >= 1 ,vai para L1
                        addi $v0,$zero,1  #retorna 1
                        addi $sp,$sp,8    #retira 2 itens da pilha  
                        jr   $ra          #retorna para depois de jal 
                     L1:addi $a0,$a0,-1   #n >= 1: argumento recebe (n-1)
                        jal  fact         #chama fact com (n-1) 
                        lw   $a0,0($sp)   #retorna de jal : restaura argumento n 
                        lw   $ra,4($sp)   #restaura o endereço de retorno 
                        addi $sp,$sp,8    #ajusta stack pointer para retirar 2 itens
                        mul  $vo,$a0,$v0  #retorna n * fact(n-1)
                        jr   $ra          #retorna para o procedimento que chamou

Considerações finais

Hoje o SIAMS não está tão robusto quanto poderia estar. Mas o projeto terá continuação e novas versões serão lançadas com melhorias na interface, desempenho e mais funcionalidades.
